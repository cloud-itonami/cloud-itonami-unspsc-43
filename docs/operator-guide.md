# Operator Guide

## First Deployment

1. Register one site.
2. Import prior inspection/operating history if available.
3. Run the advisor in read-only mode.
4. Compare recommendations with current operating practice.
5. Enable human-approved actions once the IT Asset Recovery Governor contract is trusted.

## Minimum Production Controls

- customer/municipality-owned data store
- clear provenance for every sensor reading and finding
- approval workflow for any safety-critical or public-record action
- incident contact for site operators
- monthly audit export

## Certification

Certified operators must prove data provenance, audit traceability, and
that automated actions cannot bypass the IT Asset Recovery Governor -- including proof of
whatever municipal contract or public licensing the operator's jurisdiction
requires for IT asset recovery and e-waste refurbishment work.
